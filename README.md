## Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/kushagra-aa/project_aranya.git
   ```

2. Navigate to the project directory:

   ```bash
   cd project_aranya
   ```

3. Open the `index.html` file in a web browser.
